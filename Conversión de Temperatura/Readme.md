# Conversión de Temperatura


Crea un archivo de python nuevo y modifica el siguiente código para que realize lo que se te pide:

```python
def main():
    #escribe tu código abajo de esta línea
    pass

if __name__ == '__main__':
    main()
```

La línea `#escribe tu código abajo de esta línea` es un comentario, el programa la va a ignorar al ejecutarse.

## Definición del problema

Escribe un programa que convierta una temperatura dada en escala Fahrenheit a su equivalente en grados Celsius.

El programa debe solicitar al usuario la temperatura en grados Fahrenheit (número flotante) y muéstrale el resultado de la conversion (número flotante).

Ejemplo:
```
Temperatura (F): 75.2
Conversión a grados Celsius: 24.0
```

**Nota:** No te preocupes por esta parte del código `if __name__ == '__main__':` por el momento. No la vamos a necesitar para este programa, pero es una buena práctica incluirla y quedará más claro para que sirve en los siguientes ejercicios.
